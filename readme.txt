== Description ==

 Adding post type archive link to TinyMCE link pop-up box

== Installation ==

1. Download and extract the plugin folder.
2. Upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
3. Go to Plugins screen and find "Editor archive link" in the list.
4. Click Activate to activate it.

= or =

1. Install from within your wordpress admin area by searching for "Editor archive link"
